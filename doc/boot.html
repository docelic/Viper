Viper - install, step 1/6 - boot notes


To boot, use any standard Debian install media, no need for customizations.

To boot from CD-Rom, use ready-made Debian CD/DVD images.

To boot from USB stick, the easiest way is to install GUI program "unetbootin" and use it to create a bootable stick.
