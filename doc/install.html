Viper - install, step 4/6 - install notes

The install procedure will just go on as usual, and there's not much to say about it.

Here are a couple notes that come to mind though:

Troubleshooting list

  1. Preseed file not found - see preseed.html
  2. Preseed file contains errors - see Debugging in preseed.html
  3. Install asks a question interatively - you did not preseed everything, or you preseeded a wrong key (coming back to "you did not preseed everything").
  4. Install complains about not being able to find root partition - you did not configure the disk properly in question partman-auto/disk, should place /dev/hda, /dev/sda or similar in there
